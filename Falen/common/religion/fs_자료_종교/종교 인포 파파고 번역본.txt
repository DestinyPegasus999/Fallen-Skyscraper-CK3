게임 시작 시 이 폴더에 정의된 종교와 믿음이 존재합니다.

== 구조 ==
religion_name = {
family = family_name # 종교가 속한 종교 가족은 무엇입니까?
graphical_dists = catholic_gfx # 이 종교에 대한 모든 신앙은 이 3D 모델(현재 사원 자산에 사용됨)로 기본 설정됩니다. 우선 순위는 신앙 > 종교 > 가족에서 대본으로 작성된 가장 낮은 단계이다.
경건_icon_group = 기독교 # 이 종교의 모든 신앙은 이 경건 아이콘 집합으로 기본 설정됩니다. 우선 순위는 신앙 > 종교 > 가족에서 대본으로 작성된 가장 낮은 단계이다.
trinary_background_icon = # 이 종교에 대한 모든 신앙은 기본적으로 이 교리 배경 아이콘으로 설정됩니다. 우선 순위는 신앙 > 종교 > 가족에서 대본으로 작성된 가장 낮은 단계이다.
figan_roots = yes # 만약 그렇다면, 개혁되지 않은 교리가 없는 믿음은 인터페이스에 의해 개혁된 것으로 간주된다.
교리 = 이름
교리 = another_name # 종교에 정의된 교리는 종교 내의 모든 신앙에 적용됩니다. 이것은 단지 게임 시작에 불과하다; 그것은 순전히 대본의 편리함을 위한 것이며, 모든 신앙에 교리를 두는 것과 같을 것이다. 그것은 특정 신앙에 있어서 집단에 다른 교리를 넣음으로써 무시될 수 있다. 한 가지 이상의 선택을 허용하는 교리는 종교 수준에서 정의될 수 있습니다. 종교 수준에서 정의될 수 있는 명백한 재정의 시스템은 없습니다.
# 교리는 신앙 섹션 뒤에 정의할 수 없습니다.

특성 = {
덕목 = {용감한 너그러운} # 모든 추종자를 위한 덕목인 특성을 나열하십시오. 특성 그룹도 작동합니다. 그룹에 둘 이상의 특성이 정의된 경우(또는 그룹 자체), 첫 번째 특성만 UI에 표시됩니다.
죄 = {... } # (계속)
# 미덕과 죄악은 의견의 보너스/벌칙을 부여한다(GRODURE_TRAIT 및 SINFUL_TRAIT 정의 참조). 그 점에서 중요한 것은 "시청자의" 믿음이다.
# 예를 들어, 관대한 것이 기독교의 미덕이라면, 모든 기독교의 등장인물들은 다른 사람들이 기독교인이 아니더라도, 그 특성으로 다른 모든 사람들을 더 높이 평가할 것이다.
# 특성 보유자는 각 일치 특성에 대한 덕목_owner_modifier/sin_owner_modifier도 얻는다.
# 미덕과 죄악은 선택적으로 효과를 스케일링할 승수를 가질 수 있다(기본값: 1).
덕목 = {brave = 0.5} # 의견 효과와 수식어의 스케일 조정
# 사용자 지정 한정자를 지정할 수 있습니다(기본값: 덕트_owner_modifier/sin_owner_modifier).
sins = {conflict = {monthly_class = -0.1} } }
# 사용자 지정 한정자를 사용할 때 척도(기본값: 1)도 지정할 수 있습니다.
sinces = {noberty = {monthly_discount = -0.1 척도 = 2 } # 의견 효과와 수식어 모두 척도화
}

reserved_male_names = {... } # 여기에 나열된 이름은 reserved_male_names 자체를 정의하지 않는 모든 믿음에 적용됩니다.
reserved_female_names = {... }
custom_discount_filename = {name2 name3} # 사용자 정의 믿음을 생성할 때 사용할 수 있는 아이콘이 됩니다. 경로는 "gfx/인터페이스/아이콘/종교/%s.dds"이며, 여기서 %s은 이름이다. 텍스트 아이콘도 필요합니다.

현지화 = {... } # 아래 신앙 내부의 현지화를 참조하십시오.

# 이 종교의 성스러운 명령에 의해 사용될 수 있는 이름과 CoA. 이것들은 신앙에 사용할 수 있는 것이 없을 때 사용된다.
# 여기에 남아 있지 않으면 "holy_order_default"를 이름으로 사용하고 대신 무작위로 생성된 CoA를 사용한다.
holy_order_names = {

}

holy_order_maa = {연대_type_1연대_type_2... }
